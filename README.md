# SCT_DA_4 - Business Insights Report (EDA)

## Project Overview

This project was completed as part of the SkillCraft Technology Data Analyst Internship.

The objective was to perform Exploratory Data Analysis (EDA) on a Marketing Campaign dataset to identify customer behavior, campaign effectiveness, and business insights.

## Tools Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset

Marketing Campaign Dataset

## Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Statistical Summary
- Correlation Analysis
- Income Distribution
- Age Distribution
- Product Spending Analysis
- Campaign Acceptance Analysis

---

## Business Insights

- Customers with higher income generally spend more.
- Wine products generated the highest customer spending.
- Customers aged between 30–50 years showed higher purchasing behavior.
- Some marketing campaigns achieved better customer response than others.
- Businesses should allocate more budget to the highest-performing campaigns.
